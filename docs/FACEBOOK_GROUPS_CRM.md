# Facebook Groups CRM

Goal:
Track which Facebook groups produce useful outcomes for Kromi.

CRM columns:
- Group Name
- Facebook URL
- Segment
- Priority
- Review Potential
- Kromi Fit
- ARC Friendly
- Status
- Membership Status
- Tracking ID
- Tracking URL
- Join Requested Date
- Check Again Date
- Post Result
- Post Date
- Post URL
- Post Attempt Count
- GA4 Visitors
- ARC Signups
- Amazon Clicks
- Reviews Generated
- Outcome Score
- Last Checked
- Last Error
- Notes

Daily workflow:
1. Check groups awaiting approval.
2. Update membership status.
3. Select next groups by priority.
4. Post only where allowed.
5. Update CRM.
6. Generate report.
