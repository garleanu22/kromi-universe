# Kromi Visual Style Lock

This document defines the preferred visual direction for Kromi.

## Master direction

The preferred style is premium emotional children’s book illustration.

Core qualities:

- muted pastel palette
- textured pencil or pastel rendering
- soft atmospheric depth
- subtle directional lighting
- restrained saturation
- natural emotional expressions
- premium printed children’s book aesthetic
- cinematic negative space
- gentle edge transitions
- traditional illustration feel

## Avoid

- glossy AI rendering
- excessive blur or glow
- oversaturated orange haze
- generic baby cartoon proportions
- inconsistent Kromi proportions
- low visual hierarchy
- aggressive cyan and orange contrast

## Emotional target

The illustration should communicate:

- gentle wonder
- emotional warmth
- quiet connection
- curiosity
- safety
- kindness

## Hierarchy

The eye should move naturally through:

1. main emotional subject
2. Kromi
3. supporting environment
4. background atmosphere

## Usage

Use this document as the style reference for:

- book interiors
- social media visuals
- landing page graphics
- ARC assets
- future Kromi Universe visuals
