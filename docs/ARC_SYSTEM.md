# ARC System

Goal:
Turn qualified readers into honest Amazon reviews for Kromi and the Leafless Tree.

Flow:
Facebook group post -> tracked ARC landing page -> Kit signup -> ARC delivery -> review follow-up.

Primary tracking URL format:
https://kromiuniverse.com/arc?group=fb_001

Required data:
- group parameter
- subscriber email
- ARC signup date
- follow-up status
- review status
