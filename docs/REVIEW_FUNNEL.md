# Kromi Review Funnel

Goal:
Turn targeted Facebook group traffic into ARC readers and honest Amazon reviews.

## Funnel

Facebook Group Post
↓
Tracked ARC Landing Page
↓
Kit Signup
↓
ARC Delivery
↓
Review Reminder
↓
Amazon Review

## Core URL format

https://kromiuniverse.com/arc?group=fb_001

Each Facebook group receives a unique Tracking ID.

## Metrics

Track only outcome metrics:

- Visitors
- ARC Signups
- Amazon Clicks
- Reviews Generated
- Visitor to ARC Signup Rate
- ARC to Review Rate
- Outcome Score

Do not track reach or impressions.

## Core question

Which Facebook groups generate actual ARC signups and reviews?

## Daily report sections

- Top ARC Sources
- Top Amazon Sources
- Top Review Sources
- Groups Awaiting Approval
- Groups Ready To Post
- Groups Producing No Results

## Outcome Score

- Join requested = 2
- Joined = 4
- Posted = 7
- At least one Amazon click = +1
- Each ARC signup = +1, max +3
- Each review generated = +3
